# rock_paper_scissors_spock_lizard

Building a python based rock paper scissors game with inspiration from big bang theory

Currently a work in progress, my goal is to make a fun version of rock paper scissor spock lizard that users can use to have fun and for all to enjoy. 

Note: AI tools were used as a tutor and code-review partner. The project was designed, tested, adapted, and developed as part of my Python learning practice.
